4. 头部导航栏中间有竖线的多种写法

  • 选择中间的 li 来模拟竖线
    <nav class="nav">
  <ul class="clearfix">
    <li><a href="#">首页</a></li>
    <li></li>
    <li><a href="#">发现音乐</a></li>
    <li></li>
    <li><a href="#">我的音乐</a></li>
    <li></li>
    <li><a href="#">我的音乐我的最爱哈哈</a></li>
  </ul>
</nav>

    .nav {
  float: right;
  background-color: pink;
}
  
.nav li {
  float: left;
}
  
.nav li:nth-child(even) {
  width: 1px;
  height: 16px;
  background-color: black;
  margin-top: 33px;
}
  
.nav a {
  display: block;
  /* height: 80px; */
  padding: 0 20px;
  line-height: 80px;
}

  • 直接用 span 标签包裹竖线
    <nav class="nav">
  <a href="#">首页</a><span>|</span>
  <a href="#">发现音乐</a><span>|</span>
  <a href="#">我的音乐</a><span>|</span>
  <a href="#">我的音乐我的最爱哈哈哈</a>
</nav>

    .nav {
  float: right;

  line-height: 80px;

  • 用伪元素做一个盒子来模拟竖线，再通过绝对定位定位到后面

5. 两栏布局（以左边固定宽度，右边自适应为例）

html结构均如下：

<div class="box">
    <div class="left"></div>
    <div class="right"></div>
</div>
  • 利用 BFC
    .left {
    float: left;
    width: 200px;
    height: 200px;
}
  
.right {
    /* overflow:hidden触发了BFC，这个独立的盒子不会和float的盒子发生重叠 */
    overflow: hidden;
    height: 200px;
}

  • 利用 calc() 函数
    .left {
    float: left;
    width: 200px;
    height: 200px;
}
  
.right {
    float: left;
    height: 200px;
    /* 操作符左右两侧最好写上空格，不然容易出错 */
    width: calc(100% - 200px);
}

6. 三栏布局（以两边固定宽度，中间自适应为例）

html 结构均如下：

<div class="box">
    <div class="left"></div>
    <div class="center"></div>
    <div class="right"></div>
</div>
  • 浮动float

    核心：父元素不设置定位，三列子元素都设置为左浮动。 自适应的那一列宽度用计算得出：calc(100% - 左边宽度 - 右边宽度)。

    .left {
    float: left;
    width: 200px;
    height: 200px;
}
  
.center {
    float: left;
    width: calc(100% - 400px);
    height: 200px;
}
  
.right {
    float: left;
    width: 200px;
    height: 200px;
}

  • position定位

    核心：将父元素设置为相对定位 position: relative; 左边列(固定宽度)：position: absolute; left: 0; 中间列（自适应列）：标准流 右边列（固定宽度）：position: absolute; right: 0;

    .box {
    position: relative;
    /*父元素设置相对定位*/
    padding: 0 200px;
}
  
.left {
    position: absolute;
    /*子元素绝对定位*/
    top: 0;
    left: 0;
    /*与左边距离为0*/
    width: 200px;
    height: 200px;
}
  
.center {
    height: 200px;
}
  
.right {
    position: absolute;
    /*子元素绝对定位*/
    top: 0;
    right: 0;
    /*与右边距离为0*/
    width: 200px;
    height: 200px;
}

  • flex弹性盒布局

    核心：将父元素设置为弹性盒布局：display: flex;

    自适应的那一列设置为 flex: 1

    .box {
    display: flex;
    /*父元素设置为弹性盒*/
}
  
.left {
    width: 200px;
    /*宽度固定*/
    height: 200px;
}
  
.center {
    flex: 1;
    height: 200px;
}
  
.right {
    width: 200px;
    /*宽度固定*/
    height: 200px;
}

7. 绝对、固定定位且宽高不固定的盒子水平垂直居中

html 结构均如下：

<div class="box"></div>

  • 核心：left: 0; right: 0; top: 0; bottom: 0; margin: auto;

    .box {
    position: absolute;
    /* position: fixed; */
    left: 0;
    right: 0;
    top: 0;
    bottom: 0;
    margin: auto;
    max-width: 980px;
    min-width: 320px;
    width: 100%;
    height: 100%;
}

  • 核心：left: 50%; top: 50%; transform: translate(-50%, -50%);

    .box {
    position: absolute;
    /* position: fixed; */
    left: 50%;
    top: 50%;
    max-width: 980px;
    min-width: 320px;
    width: 100%;
    height: 100%;
    transform: translate(-50%, -50%);
    /* 也是向左向上移动当前元素宽度的一半 */
}

8. background-size 属性

在使用精灵图或其它背景图时我们时常遇到二倍图，需要把它缩小

  • 值的个数

    • 一个值

      这个值指定图片的宽度，此时图片高度默认为 auto

    • 两个值

      第一个值指定图片的宽度，第二个值指定图片的高度

  • 值的种类

    • auto

      背景图片与插入图片一样，会随着另一个非 auto 值的改变而等比例缩放

    • 具体值（px、em等）

      背景图片把自身按照具体值大小缩放

    • 百分比

      这个百分比是相对于当前盒子的大小来说的，所以背景图片按照当前盒子的宽高百分比缩放

    • contain

      缩放背景图片以完全装入容器，可能背景区部分空白。尽可能的缩放背景并保持图像的宽高比例（图像不会被压缩）。当背景图和容器的大小的不同时，容器的空白区域（上/下或者左/右）会显示由 background-color 设置的背景颜色。

    • cover

      缩放背景图片以完全覆盖容器，可能背景图片部分看不见。尽可能大的缩放背景图像并保持图像的宽高比例（图像不会被压扁）。当容器和背景图大小不同时，背景图的 左/右 或者 上/下 部分会被裁剪。

      （比如此例我们就看不到小姐姐了呜呜呜。。。）
