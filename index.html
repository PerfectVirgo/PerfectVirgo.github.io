HTML入门笔记1

1. HTML历史 HTML是在1990年左右诞生,由英国人Tim Berners-Lee发明，与此同时他还发明了HTTP和URL，与HTML共同组成了WWW（万维网） 2. HTML起手应该写什么 <!DOCTYPE html> <!-- 声明文档类型为html --> <html lang="zh-CN"> <!-- 声明语言为中文-中国 --> <head> <meta charset="UTF-8"> <!-- 声明字符集为UTF-8 --> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 声明视口禁用缩放，以兼容手机 --> <meta http-equiv="X-UA-Compatible" content="ie=edge"> <!-- 声明如果当前页面在IE浏览器里显示，IE请升级成最新的内核 --> <title>Document</title> <!-- 页面标题 --> </head> <body> </body> </html> 3. 常用的章节标签 标题 h1~h6 章节 section 文章 article 段落 p 头部 header 脚部 footer 主要内容 main 旁支内容 aside 划分 div 4. 全局属性 class——给标签分一个类

如何搭建个人博客

如何用hugo搭建个人博客 首先这是hugo官方教程，当然如果你不喜欢看英文的话，那就往下看我的教程吧 安装hugo 在https://github.com/gohugoio/hugo/releases里下载安装包并解压验证安装，显示版本号即安装成功 $ hugo version 建立新网站　 将quickstar替换为你用于存放和上传博客的本地仓库的文件夹名称 $ hugo new site quickstart 添加主题　 首先，从Github下载默认主题并将其添加到您网站的theme目录中 $ cd quickstart $ git init $ git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke 然后，将主题添加到站点配置中 $ echo 'theme = "ananke"' >> config.toml 开始写博客 首先创建一个新.md文件 $ hugo new posts/my-first-post.md 新创建的文件会从以下内容开始 --- $ title: "My First Post" $ date: 2019-03-26T08:47:11+01:00 $ draft: true --- 这里要注意写完博客后要把draft: true修改为draft: false，因为draft是草稿的意思
