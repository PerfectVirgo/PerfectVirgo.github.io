1. a 标签的用法

  • href

    取值：

    • 网址

      https://google.com

      http://google.com

      //google.com 最高级，它会自动选择是用https还是http

    • 路径

      /a/b/c 以及 a/b/c 注意绝对路径开头的/是指http-server的根目录，而不是指硬盘的根目录

      index.html 以及 ./index.html

    • 伪协议

      javascript:代码; 用法：当你想点击一个a链接什么都不做的话，就让href="javascript:;"

      ​ 那如果href=""呢？结果是点击这个a链接页面会刷新

      ​ 那如果href="#"呢？结果是点击这个a链接页面会自动滚到顶部

      mailto:邮箱 会自动跳转到发邮件的界面，并自动填写mailto后面的邮箱

      tel:手机号 会自动跳转到拨号的界面，并自动填写tel后面的手机号

    • id

      #xxx 锚点链接，页面会跳转到指定的标签

  • target

    取值：

    （1）内置名字：

    • _blank
    • _self
    • _top 如果有多级窗口，就在最顶级的窗口打开
    • _parent 如果有多级窗口，就在其父级窗口打开

    （2）程序员命名：

    • window的name,比如xxx

      如果有一个叫xxx的窗口，就用它，如果没有呢，就创建一个窗口，把它叫作xxx

    • iframe的name

  • download

    下载页面。但不是所有浏览器都支持，尤其是手机浏览器不支持

  • rel=noopener

    为了防止一个bug，等学了JS后再回来填坑

2. iframe 标签

​　　内嵌窗口，页面可以通过iframe引用另一个页面，需要注意的是google是不允许别人用iframe指向它的。不过已经很少使用了，还有些老系统在用。

5. img 标签的用法

  • 作用

    发出 get 请求，展示一张图片

  • 属性

    alt / height / width / src

  • 事件

    onload / onerror

  • 响应式

    max-width:100%

  • 可替换元素

6. form 标签的用法

  • 作用

    发 get 或 post 请求，然后刷新页面

  • 属性

    • action 控制请求哪个页面
    • method 控制请求方式是用GET还是POST
    • autocomplete 控制是否自动填充，on/off
    • target 和a链接的 target 用法相同，告诉浏览器我要提交到哪个页面，哪个页面需要刷新

  • 事件

    onsubmit

    Q：<input type="submit" value="搞起" />与<button type="submit">搞起</button> 有什么区别？

    A：input里面不能再放其它标签，只能有纯文本，而button里面可以有任何标签

7.input 标签的用法

  • 作用

    让用户输入内容

  • 属性

    • 类型 type: button / checkout / email / file / hidden / number / password / radio / search / submit / tel / text
    • 其它: name / autofocus / checked / disabled / maxlength / pattern / value / palceholder

  • 事件

    onchange / onfocus / onblur

  • 验证器

    HTML5 新增功能

  • 注意事项：一般不监听 input 的 click 事件
